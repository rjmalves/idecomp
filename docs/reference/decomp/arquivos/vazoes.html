Vazões por Posto (vazoes.rvX)¶

As vazões por posto do DECOMP, localizadas no arquivo geralmente denominado vazoes.rvX, são armazenadas na classe:

class Vazoes(data=Ellipsis)[código fonte]¶

Armazena os dados de entrada do DECOMP referentes ao cadastro das usinas hidroelétricas.

property vazoes: DataFrame¶

Obtém a tabela com os dados de vazão existentes no arquivo binário.

  • 1…N (int) - onde N é o número de postos

Retorna:

A tabela com as vazões por posto

Tipo de retorno:

pd.DataFrame
