Operação das Usinas Termoelétricas (dec_oper_usit.csv)¶

As informações de operação das usinas termoelétricas são armazenadas na classe:

class DecOperUsit(data=<cfinterface.data.blockdata.BlockData object>)[código fonte]¶

Arquivo com a operação por usina termoelétrica do DECOMP.

property tabela: DataFrame | None¶

A tabela de dados que está contida no arquivo.

  • periodo (int)

  • no (int)

  • cenario (int)

  • patamar (int)

  • duracao (float)

  • indiceUsina (int)

  • nomeUsina (str)

  • indiceSubmercado (int)

  • nomeSubmercado (str)

  • custoIncremental (float)

  • geracaoMinimaMW (float)

  • geracaoMW (float)

  • fatorManutencao (float)

  • geracaoMaximaMW (float)

  • custoGeracao (float)

Retorna:

A tabela como um dataframe

Tipo de retorno:

pd.DataFrame | None
