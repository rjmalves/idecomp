Sumario (sumario.rvX)¶

As informações de saída do DECOMP, localizadas no arquivo geralmente denominado sumario.rvX, onde X varia de 0 a 4, são armazenadas na classe:

class Sumario(data=<cfinterface.data.blockdata.BlockData object>)[código fonte]¶

Armazena os dados de saída do DECOMP referentes ao acompanhamento do programa.

Esta classe lida com as informações de entrada fornecidas ao DECOMP e reproduzidas no sumario.rvx, bem como as saídas finais da execução: custos de operação, despacho de térmicas, etc.

property cmo_medio_subsistema: DataFrame¶

Obtém a tabela de CMO existente no Sumario

  • Subsistema (str)

  • Patamar (str)

  • Estágio 1 (float)

  • Estágio 2 (float)

  • …

  • Estágio N (float)
