Relato (relato.rvX)¶

Visão geral do modelo¶

As informações de saída do DECOMP, localizadas no arquivo geralmente denominado relato.rvX, onde X varia de 0 a 4, são armazenadas na classe:

class Relato(dados: idecomp._utils.dadosarquivo.DadosArquivoBlocos)[código fonte]¶

Armazena os dados de saída do DECOMP referentes ao acompanhamento do programa.

Esta classe lida com as informações de entrada fornecidas ao DECOMP e reproduzidas no relato.rvx, bem como as saídas finais da execução: custos de operação, despacho de térmicas, etc.

classmethod le_arquivo(diretorio: str, nome_arquivo='relato.rv0') → idecomp.decomp.relato.Relato[código fonte]¶

Realiza a leitura de um arquivo “relato.rvx” existente em um diretório.

Parâmetros

  • diretorio (str) – O caminho relativo ou completo para o diretório onde se encontra o arquivo

  • nome_arquivo (str, optional) – Nome do arquivo a ser lido, potencialmente especificando a revisão. Tem como valor default “relato.rv0”

Retorna

Um objeto Relato com informações do arquivo lido
