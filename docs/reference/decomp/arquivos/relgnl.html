Relatório de Térmicas GNL (relgnl.rvX)¶

Visão geral do modelo¶

As informações de saída das térmicas de despacho antecipado (GNL) do DECOMP, localizadas no arquivo geralmente denominado relgnl.rvX, onde X varia de 0 a 4, são armazenadas na classe:

class RelGNL(dados: idecomp._utils.dadosarquivo.DadosArquivoBlocos)[código fonte]¶

Armazena os dados de saída do DECOMP referentes às térmicas de despacho antecipado (GNL).

Esta classe lida com as informações de entrada fornecidas ao DECOMP e reproduzidas no relgnl.rvx, bem como as saídas finais da execução.
